# Max AI

Max AI is a free multilingual AI assistant with chat, voice, and image generation capabilities powered by Claude AI.

## Features
- **Multilingual Support**: Interact in multiple languages, enabling users from different linguistic backgrounds to utilize the assistant effectively.
- **Chat Functionality**: Engage in conversational interactions, providing users with quick answers and assistance.
- **Voice Commands**: Utilize voice for commands to enhance usability and accessibility.
- **Image Generation**: Generate images based on user input, bringing creativity to life.

## How to Use
1. Clone the repository:
   ```bash
   git clone https://github.com/mmaheshkhatri2001-coder/Mahesh-Khatri.git
   ```
2. Install the required dependencies:
   ```bash
   npm install
   ```
3. Run the application:
   ```bash
   npm start
   ```
4. Interact with Max AI through the interface that appears on your browser.

## Deployment Instructions
- Ensure you have Node.js and npm installed on your machine.
- Follow the setup instructions above to get the application running locally.
- For deployment on a production server, consider using Docker or a cloud platform like AWS, Heroku, or similar services.

## API Setup
- Obtain your API keys from Claude AI and configure them in your environment variables.
- Follow the API documentation provided by Claude AI to integrate their functionalities into the Max AI application.
